I finally managed to get this site up and running, thanks to the ease with which tools like [Jekyll](http://jekyllrb.com/) have made it possible in a very short duration. What Jekyll does is transform your plain text files into static web pages -- it is a static site generator written by @mojombo.

This being my first post I am going to write about my experience setting up this website. First, I wanted to have a website that can easily be managed using version control. So that anytime am pushing my files to the server the website gets updated instantly. I without any second thoughts decided to host this site on [github](http://github.com) using [github pages](http://pages.github.com) - and what's more, it is free of charge.

Now let's look at the requirements.

I will be working on my local machine nicknamed **SIBUOR** - my native language for Mountain Lion. I require:-

+github account
+ruby 1.9.3 - since I will be using github-pages
+github-pages gem - this installs Jekyll and other requires gems
